28   ### Maven Parent overrides
29   
30   Due to Maven's design, elements are inherited from the parent POM to the project POM.
31   While most of the inheritance is fine, it also inherits unwanted elements like `<license>` and `<developers>` from the parent.
32   To prevent this, the project POM contains empty overrides for these elements.
33   If you manually switch to a different parent and actually want the inheritance, you need to remove those overrides.
